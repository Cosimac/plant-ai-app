# 图标资源目录

此目录用于存放小程序所需的图标资源。

## 需要的图标文件

- `home.png` - 首页图标
- `home-active.png` - 首页激活图标
- `camera.png` - 相机图标
- `camera-active.png` - 相机激活图标
- `history.png` - 历史图标
- `history-active.png` - 历史激活图标
- `profile.png` - 个人中心图标
- `profile-active.png` - 个人中心激活图标

## 图标规格

- 尺寸：建议 81x81 像素
- 格式：PNG
- 背景：透明
- 颜色：未激活状态为灰色，激活状态为主题色

## 获取图标

可以从以下途径获取图标：
1. 使用图标库如 iconfont
2. 设计工具如 Figma、Sketch
3. 在线图标生成器
